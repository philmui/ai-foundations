Module 5: Working with Structured Data

Learn to manipulate data like a programmable spreadsheet

What is
Structured Data?

Data organized in rows and columns, like a table

  • Each row represents an observation
  • Each column represents a variable
  • Perfect for analysis and machine learning

Pandas: Your
Programmable
Spreadsheet

Excel meets Python: all the power of spreadsheets,
with the flexibility of code

Loading Data

Import data from various sources

import pandas as pd

# Load CSV file
df = pd.read_csv('data.csv')

# Load JSON file
df = pd.read_json('data.json')

# Load from URL
df = pd.read_csv('https://example.com/data.csv')

# Load with specific encoding
df = pd.read_csv('data.csv', encoding='utf-8')
  • pd.read_csv() — most common format
  • pd.read_json() — for JSON data
  • pd.read_sql() — from databases
  • pd.read_excel() — from Excel files

What .info() reveals

  • Total entries and column count
  • Non-null counts (spotting missing data)
  • Data types for each column
  • Memory usage estimate

.describe()

Statistical summary of numeric columns

  • count — number of non-null values
  • mean — average value
  • std — standard deviation (spread)
  • quartiles — 25%, 50% (median), 75%

Finding Missing Data

Identify gaps in your dataset

  • df.isna() — Boolean mask of missing values
  • df.isna().sum() — Count per column
  • df.dropna() — Remove rows with NaN
  • df.fillna() — Replace NaN with a value

Selecting &
Filtering

Extract exactly what you need

# Select specific columns
df[['model', 'rating', 'category']]

# Select single column (Series)
df['model']

# Filter rows by condition
df[df['rating'] > 4.5]

# Multiple conditions
df[(df['category'] == 'coding') & (df['rating'] >= 4.7)]

# Select by position
df.iloc[0:10]   # First 10 rows
df.iloc[0, 3]   # Row 0, Column 3

# Select by label
df.loc[df['model'] == 'gpt-4']

Grouping Data

Aggregate and summarize by category

# Group by model and calculate mean rating
df.groupby('model')['rating'].mean()

# Multiple aggregations
df.groupby('category').agg({
    'rating': ['mean', 'count'],
    'tokens_used': ['min', 'max']
})

Connection to
Machine Learning

Why clean data inspection matters

Data Quality Impacts Model Performance

  • Missing values → must impute or remove
  • Wrong data types → conversion errors
  • Outliers → skewed predictions
  • Class imbalance → biased models

Inspection Guides Preprocessing

  • Categorical columns → need encoding
  • Numeric ranges → may need scaling
  • Text columns → require vectorization
  • Timestamps → extract features (hour, day)
